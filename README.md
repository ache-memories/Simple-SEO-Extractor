# 🚀 Simple-SEO-Extractor

A lightweight, high-speed Bash tool designed for metadata extraction and text normalization. Perfect for SEO managers, media enthusiasts, and Linux power users who need clean data without the bloat of heavy browser automation.

## ✨ Features
- **Fast Extraction**: Grabs Title, Cast (Starring), and Description in seconds.
- **Text Normalization**: Automatically cleans HTML entities and standardizes special characters.
- **SEO Ready**: Generates SEO-friendly slugs for URLs and file naming.
- **Zero Bloat**: Written in pure Bash using `curl` and `grep`.
